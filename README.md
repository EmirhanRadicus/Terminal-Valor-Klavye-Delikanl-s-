# Terminal-Valor-Klavye-Delikanl-s-
Klavye Delikanlıları, HTML, CSS, JS ve jQuery ile geliştirilen, "Welcome to the Game" esintili web tabanlı bir hacking simülasyonudur. 3 kişilik bir ekiple hazırlanan bu okul projesi; interaktif terminali, sürükleyici hikayesi ve siber operasyon atmosferiyle tarayıcı üzerinden derinlemesine bir rol yapma ve hacking deneyimi sunmayı hedefliyoruz
